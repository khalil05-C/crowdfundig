{% if is_ar %}كيف تعمل حملة التمويل؟{% else %}Comment fonctionne une cagnotte ?{% endif %}

{% if is_ar %} الحملة تسمح لك بعرض مشروعك، تحديد هدف مالي بالدرهم، واستقبال مساهمات من أشخاص يريدون دعم فكرتك. تصبح الحملة عامة فقط بعد موافقة المشرف. {% else %} Une cagnotte permet de présenter un projet, fixer un objectif en MAD et recevoir des contributions de personnes qui souhaitent soutenir votre idée. Sur FundRise MA, une cagnotte devient publique seulement après validation par le modérateur. {% endif %}

{% if is_ar %}1. إنشاء الحملة{% else %}1. Créer la cagnotte{% endif %}

{% if is_ar %}أضف العنوان، الوصف، الفئة، الهدف المالي، تاريخ النهاية وصورة الغلاف. يمكنك حفظها كمسودة أو إرسالها للمشرف.{% else %}Le porteur ajoute le titre, la description, la catégorie, l'objectif financier, la date de fin et une image. Il peut sauvegarder en brouillon ou soumettre au modérateur.{% endif %}

{% if is_ar %}2. المراجعة{% else %}2. Validation{% endif %}

{% if is_ar %}يتحقق المشرف من المحتوى قبل النشر. قبل الموافقة لا تظهر الحملة للزوار ولا يمكن لأي شخص المساهمة.{% else %}Le modérateur vérifie le contenu avant publication. Tant que la cagnotte n'est pas approuvée, elle ne s'affiche pas aux visiteurs et personne ne peut contribuer.{% endif %}

{% if is_ar %}3. المساهمات{% else %}3. Contributions{% endif %}

{% if is_ar %}بعد الموافقة يظهر زر الدعم. يختار المساهم المبلغ، يدفع، ويتم تحديث تقدم الحملة تلقائيا.{% else %}Une fois active, la cagnotte affiche le bouton Soutenir. Les contributeurs choisissent un montant, paient, puis la progression se met à jour automatiquement.{% endif %}

{% if is_ar %}أنواع الحملات{% else %}Types de cagnotte{% endif %}

{% if is_ar %}كل شيء أو لا شيء{% else %}Tout ou rien{% endif %}

{% if is_ar %}تستخدم الأموال إذا تم بلوغ الهدف. هذا النوع مناسب للمشاريع التي تحتاج الميزانية كاملة للانطلاق.{% else %}Les fonds sont destinés au projet si l'objectif est atteint. Ce mode rassure les contributeurs quand le projet a besoin du budget complet pour démarrer.{% endif %}

{% if is_ar %}مرن{% else %}Flexible{% endif %}

{% if is_ar %}يمكن لصاحب المشروع التقدم بالأموال التي تم جمعها حتى إذا لم يتحقق الهدف الكامل.{% else %}Le porteur peut avancer avec les fonds collectés même si l'objectif total n'est pas atteint. Ce mode convient aux projets qui peuvent progresser par étapes.{% endif %}

  • {% if is_ar %}حدد هدفا واقعيا وتاريخ نهاية واضحا.{% else %}Fixez un objectif réaliste et une date de fin crédible.{% endif %}
  • {% if is_ar %}انشر تحديثات منتظمة للمساهمين.{% else %}Partagez régulièrement des mises à jour aux contributeurs.{% endif %}
